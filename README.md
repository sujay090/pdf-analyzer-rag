# PDF RAG Analyzer

A full-stack application that allows users to upload PDF documents and ask questions about them using Retrieval-Augmented Generation (RAG).

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS, Clerk (Authentication)
- **Backend**: Node.js, Express, BullMQ (Task Queue), LangChain
- **Vector Database**: Qdrant
- **AI Models**: Google Gemini (Embeddings & Generation)
